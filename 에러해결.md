### 2. 브라우저 캐시 클리어

1. **강력 새로고침:**
   - `Ctrl + Shift + R` (Windows)
   - 또는 `Ctrl + F5`

2. **개발자 도구에서:**
   - F12 → Network 탭
   - "Disable cache" 체크
   - 페이지 새로고침

3. **브라우저 캐시 완전 삭제:**
   - Chrome: 설정 → 개인정보 및 보안 → 인터넷 사용 기록 삭제
   - Edge: 설정 → 개인정보, 검색 및 서비스 → 인터넷 사용 기록 삭제

### 3. Vite 캐시 삭제

```powershell
# Vite 캐시 삭제
Remove-Item -Path "node_modules\.vite" -Recurse -Force
```
