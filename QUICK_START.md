# 빠른 시작 가이드

## 🚀 현재 상태

- ✅ 데이터베이스 생성 및 데이터 import 완료
- ✅ 백엔드 API 서버 실행 중 (`http://localhost:3000`)
- ✅ 데이터베이스 연결 확인 완료

## 📋 다음 단계 (순서대로 진행)

### Step 1: 환경 변수 파일 생성

프로젝트 루트에 `.env` 파일 생성:

```env
VITE_API_BASE_URL=http://localhost:3000/api
VITE_USE_MOCK_DATA=false
```

### Step 2: 프론트엔드 재시작

```bash
# 현재 실행 중인 프론트엔드 서버 중지 (Ctrl+C)
npm run dev
```

### Step 3: API 테스트

브라우저에서 다음 URL 테스트:

1. **능력단위 검색:**
   ```
   http://localhost:3000/api/ability-units?keyword=품질
   ```

2. **능력단위 상세 (KSA 포함):**
   ```
   http://localhost:3000/api/ability-units/0101010101_17v2
   ```

3. **기관 목록:**
   ```
   http://localhost:3000/api/organizations
   ```

### Step 4: 프론트엔드에서 확인

1. 홈 페이지에서 검색 테스트
2. 상세 페이지에서 KSA 데이터 확인
3. 실제 DB 데이터가 표시되는지 확인

## 🐛 문제 해결

### API가 응답하지 않음
- 백엔드 서버가 실행 중인지 확인
- `http://localhost:3000` 접속 확인

### 프론트엔드에서 Mock 데이터가 표시됨
- `.env` 파일의 `VITE_USE_MOCK_DATA=false` 확인
- 프론트엔드 서버 재시작 확인

### CORS 오류
- 백엔드 서버의 CORS 설정 확인 (이미 설정되어 있음)
- 브라우저 콘솔 확인
