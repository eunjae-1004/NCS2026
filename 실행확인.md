# 실행 화면 확인 가이드

## 현재 상태

### ✅ 프론트엔드 서버
- **포트**: 5173
- **상태**: 실행 중
- **접속**: http://localhost:5173

### ⚠️ API 서버
- **포트**: 3000
- **상태**: 확인 필요

## 확인 방법

### 1. 브라우저에서 확인

**프론트엔드 접속:**
```
http://localhost:5173
```

**API 서버 테스트:**
```
http://localhost:3000
```

### 2. PowerShell에서 확인

```powershell
# 포트 확인
netstat -ano | findstr ":3000 :5173" | findstr "LISTENING"

# API 서버 테스트
Invoke-RestMethod -Uri http://localhost:3000

# 프론트엔드 테스트
Invoke-WebRequest -Uri http://localhost:5173 -UseBasicParsing
```

## 브라우저에서 보이는 화면

### 정상 작동 시

1. **로그인 페이지** (`http://localhost:5173/login`)
   - "NCS 검색 시스템" 제목
   - 이름 입력 필드
   - 기관 선택 드롭다운
   - 로그인/건너뛰기 버튼

2. **홈 대시보드** (`http://localhost:5173`)
   - "무엇을 하시겠어요?" 제목
   - 3개의 큰 카드:
     - 직무로 찾기 (파란색)
     - 키워드로 찾기 (초록색)
     - 추천으로 시작하기 (보라색)

### 문제 발생 시

1. **빈 화면**
   - 브라우저 콘솔 확인 (F12)
   - 에러 메시지 확인

2. **"Cannot GET /" 에러**
   - 프론트엔드 서버 재시작

3. **연결 오류**
   - 서버가 실행 중인지 확인

## 빠른 확인 명령어

```powershell
# 모든 서버 상태 한번에 확인
Write-Host "=== 서버 상태 ===" -ForegroundColor Cyan
Write-Host "API 서버 (3000):" -NoNewline
if (netstat -ano | findstr ":3000" | findstr "LISTENING") { 
    Write-Host " ✅ 실행 중" -ForegroundColor Green 
} else { 
    Write-Host " ❌ 실행 안 됨" -ForegroundColor Red 
}
Write-Host "프론트엔드 (5173):" -NoNewline
if (netstat -ano | findstr ":5173" | findstr "LISTENING") { 
    Write-Host " ✅ 실행 중" -ForegroundColor Green 
} else { 
    Write-Host " ❌ 실행 안 됨" -ForegroundColor Red 
}
```
